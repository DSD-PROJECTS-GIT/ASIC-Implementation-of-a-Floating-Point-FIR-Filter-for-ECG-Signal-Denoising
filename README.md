# ASIC-Implementation-of-a-Floating-Point-FIR-Filter-for-ECG-Signal-Denoising
This project presents the design and implementation of a floating-point based Finite
Impulse Response (FIR) filter using Verilog Hardware Description Language (HDL).
The system is developed based on the IEEE 754 single-precision floating-point
standard to ensure high accuracy in digital signal processing applications.
The design includes custom implementation of floating-point arithmetic units such
as addition and multiplication modules. These modules are integrated to realize
a pipelined FIR filter architecture capable of processing real-time input signals
efficiently. The filter design is optimized using pipelining techniques to improve
performance and throughput.
The proposed system is simulated and verified using hardware design tools, ensuring
correct functionality and timing behavior. The results demonstrate that the floating-
point FIR filter provides improved precision compared to fixed-point implementations,
making it suitable for applications such as biomedical signal processing.
Keywords: Floating Point, IEEE 754, FIR Filter, Verilog HDL, Pipelining, Digital Signal Processing
